# Local static server

This project includes a minimal, zero-dependency Node static server to serve the site locally.

Run:

PowerShell (Windows):

```powershell
$env:PORT=3000  # optional
npm start
```

Command prompt (Windows):

```cmd
set PORT=3000
npm start
```

Or run directly:

```bash
node server.js
```

Then open http://localhost:8080 (or the port you set).
